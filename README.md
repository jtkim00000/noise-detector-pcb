## Overview

The following documentation is a full explaination and walk through of my Noise Detector PCB, an analog circuit. 

The circuit includes an analog signal chain (amplification → peak detection → timed delay → threshold comparison). The components include a condesnser microphone, LM358 op-amp, LM311 comparator, and a Schmitt Trigger Inverter. 

The design includes an NMOS-switched peak detector, an RC hold time delay circuit to capture audio peak for a set time, and a 3-level voltage comparator network that connects to a 3-level LED volume indicator.
